## ⚠️ 常见问题

### 问题 1: 编译报错
**原因**: 页面路径不存在或 app.json 配置错误

**解决**: 
- 检查页面文件是否存在
- 检查 app.json 中 pages 数组是否包含新页面

### 问题 2: 网络请求失败
**原因**: 服务器域名未生效或 API 服务未启动

**解决**:
- 确认小程序后台域名配置已保存
- 重新编译小程序
- 检查服务器 API 服务状态

### 问题 3: 真机预览扫码失败
**原因**: 网络问题或权限问题

**解决**:
- 确认手机和电脑在同一网络
- 确认开发者工具有权限
- 重启开发者工具
