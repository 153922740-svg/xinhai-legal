3. 点击"获取验证码"按钮
4. 查看提示的验证码（开发模式会弹窗显示）

### 步骤 2: 登录

1. 输入手机号

## ⚠️ 常见问题

### 问题 1: 点击获取验证码没反应
**原因**: 手机号格式不正确  
**解决**: 确保输入 11 位手机号

### 问题 2: 登录失败提示"验证码错误"
**原因**: 验证码已过期或不匹配  
**解决**: 重新获取验证码，使用最新验证码

### 问题 3: 网络错误
**原因**: 服务器域名未配置或 API 服务未启动  
**解决**: 
- 检查小程序后台域名配置
- 检查 API 服务是否运行
