常见问题解答

💬 咨询类

AI 法律咨询准确吗？

+
心海法律 AI 采用三模型交叉验证技术（DeepSeek + Qwen + GLM），法律条文准确率可达 99.2%。但 AI 建议仅供参考，复杂案件建议咨询专业律师。

咨询如何收费？

+
AI 咨询按 Token 消耗计费，根据问题复杂度和回答长度实时扣费。平均每次咨询消耗约 100-500 Token（约 0.02-0.1 元）。

可以上传文件分析吗？

+
支持上传 PDF、Word、TXT 文件，AI 会自动提取文本内容并分析。也支持图片上传，通过 OCR 识别文字后分析。

对话记录会保存吗？

+
是的，您的所有对话记录都会加密保存，可在"历史对话"中查看。除非您主动删除，否则永久保存。

👑 会员类

会员有什么用？

+
会员是功能使用权限，开通会员后可使用文书生成、合同审阅等核心功能。会员不含免费服务次数，使用 AI 服务仍需消耗 Token。

会员有哪些套餐？

+
• 新人福利：注册送 3 天免费会员
• 连续包月：首月¥1，次月¥30/月
• 月卡：¥30/30 天
• 季卡：¥80/90 天
• 年卡：¥288/365 天

会员到期了怎么办？

+
会员到期后，您将无法继续使用文书生成、合同审阅等会员功能。如需继续使用，可重新开通会员。

如何取消自动续费？

+
在微信支付的自动续费管理中可随时取消。路径：微信 - 我 - 服务 - 钱包 - 支付设置 - 自动续费。

📄 文书类

支持哪些文书类型？

+
支持 9 种常用法律文书：民事起诉状、答辩状、律师函、租房合同、借款协议、离婚协议、劳动仲裁申请书、债权转让协议、和解协议。

生成的文书可以直接使用吗？

+
AI 生成的文书可作为参考模板，建议根据实际情况修改完善。重要法律文书建议咨询专业律师审核后使用。

文书可以下载吗？

+
支持下载 Word 和 PDF 两种格式，可在文书详情页点击"下载"按钮。

🪙 Token 类

Token 是什么？

+
Token 是平台虚拟服务计费单位，用于支付 AI 咨询、文书生成等服务的消耗。官方定价：2 元 = 10,000 Token。

Token 可以退款吗？

+
Token 为虚拟服务计费单位，一经充值，不可提现、不可退款、不可转让。

🤝 合伙人类

如何成为合伙人？

+
所有用户注册后即可成为普通合伙人，生成专属推荐码。推荐好友注册、开通会员、充值 Token 均可获得佣金奖励。

👤 账号类

如何更换手机号？

+
在个人中心 - 编辑资料中可以修改绑定手机号。需要验证原手机号和新手机号。
