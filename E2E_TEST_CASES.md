## 常见问题排查

### 问题 1: 登录失败，控制台显示"不在 request 合法域名列表"

**原因**: 微信公众平台未配置域名

**解决**:
1. 登录 mp.weixin.qq.com
2. 开发 → 开发管理 → 开发设置 → 服务器域名
3. 添加 https://xinclaw.xhacca.cn 到 request 合法域名
4. 保存后重新编译小程序

### 问题 2: 登录后仍然显示登录页面

**原因**: token 未正确保存或读取

**排查**:
1. 检查 login.js 中 wx.setStorageSync('token', res.data.token)
2. 检查 app.js 中 wx.getStorageSync('token')
3. 检查控制台是否有报错

### 问题 3: 验证码发送失败

**原因**: API 接口问题或网络问题

**排查**:
1. 检查控制台网络请求
2. 检查 API 服务日志
3. 检查 Nginx 配置
