## 🔧 测试步骤

### Step 1: 小程序登录
```javascript
// 获取用户 openid
wx.login({
  success: (res) => {
    // 调用后端接口换取 openid
    wx.request({
      url: 'https://xinclaw.xhacca.cn/api/v2/payment/get_openid',
      method: 'POST',
      data: { code: res.code },
      success: (res) => {
        const openid = res.data.openid
        console.log('用户 openid:', openid)
      }
    })
  }
})
```

**预期结果**: 成功获取用户 openid

---

### Step 2: 创建测试订单
```javascript
wx.request({
  url: 'https://xinclaw.xhacca.cn/api/v2/order/create',
  method: 'POST',
  header: {
    'Authorization': 'Bearer ' + token
  },
  data: {
    package_id: 'monthly',
    amount: 0.01,  // 测试金额 0.01 元
    product_type: 'member'
  },
  success: (res) => {
    const orderId = res.data.order_id
    console.log('订单 ID:', orderId)
  }
})
```

**预期结果**: 创建订单成功，返回订单 ID

---

### Step 3: 调用微信支付接口
```javascript
wx.request({
  url: 'https://xinclaw.xhacca.cn/api/v2/pay/wechat',
  method: 'POST',
  header: {
    'Authorization': 'Bearer ' + token
  },
  data: {
    order_id: orderId,
    openid: openid,
    amount: 0.01
  },
  success: (res) => {
    // 获取支付参数
    const payParams = res.data
    console.log('支付参数:', payParams)
    
    // 调起微信支付
    wx.requestPayment(payParams)
  }
})
```

**预期结果**: 返回支付参数（timeStamp/nonceStr/package/signType/paySign）

---

### Step 4: 小程序调起支付
```javascript
wx.requestPayment({
  timeStamp: payParams.timeStamp,
  nonceStr: payParams.nonceStr,
  package: payParams.package,
  signType: payParams.signType,
  paySign: payParams.paySign,
  success: (res) => {
    console.log('支付成功', res)
    // 跳转支付结果页
    wx.redirectTo({
      url: '/pages/pay-result/pay-result?status=success'
    })
  },
  fail: (err) => {
    console.log('支付失败', err)
    if (err.errMsg.indexOf('cancel') === -1) {
      wx.showToast({ title: '支付失败', icon: 'none' })
    }
  }
})
```

**预期结果**: 微信支付界面弹出，完成支付
