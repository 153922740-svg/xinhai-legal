   135|### 5.2 详细设计
   136|
   137|#### POST /api/v1/{endpoint}
   138|
   139|**请求头**:
   140|```
   141|Content-Type: application/json
   142|Authorization: Bearer ***
   143|```
   144|
   145|**请求体**:
   146|```json
   147|{
   148|  "field1": {
   149|    "type": "string",
   150|    "required": true,
   151|    "description": "{描述}"
   152|  },
   153|  "field2": {
   154|    "type": "integer",
   155|    "required": false,
   156|    "description": "{描述}"
   157|  }
   158|}
   159|```
   160|
   161|**响应体**:
   162|```json
   163|{
   164|  "code": 200,
   165|  "data": {
   166|    "result": "value"
   167|  },
   168|  "message": "success"
   169|}
   170|```
   171|
   172|**错误码**:
   173|| 码 | 说明 | 处理 |
   174||----|------|------|
   175|| 400 | 请求参数错误 | 返回详细错误 |
   176|| 401 | 未授权 | 跳转登录 |
   177|| 500 | 服务器错误 | 记录日志 |
