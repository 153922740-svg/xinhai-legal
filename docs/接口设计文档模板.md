## 接口设计

### 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 | 示例 |
|--------|------|------|--------|------|------|
| param1 | string | 是 | - | 参数说明 | "value" |
| param2 | int | 否 | 0 | 参数说明 | 123 |

### 请求示例

```json
{
  "param1": "value",
  "param2": 123
}
```

---

### 响应格式

**成功响应**:
```json
{
  "code": 200,
  "data": {
    "field1": "value1",
    "field2": "value2"
  },
  "message": "success"
}
```

**错误响应**:
```json
{
  "code": 400,
  "data": null,
  "message": "参数错误"
}
```

---

### 错误码

| 错误码 | HTTP 状态码 | 说明 | 处理建议 |
|--------|-----------|------|----------|
| success | 200 | 成功 | - |
| invalid_params | 400 | 参数错误 | 检查请求参数 |
| auth_required | 401 | 需要登录 | 添加 Token |
| server_error | 500 | 服务器错误 | 联系管理员 |
