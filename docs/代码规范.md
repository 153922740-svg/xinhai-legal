## 命名规范

### 变量命名

```python
# ✅ 正确
user_name = "张三"
total_count = 100
is_valid = True
MAX_RETRY = 3

# ❌ 错误
username = "张三"  # 缺少下划线
totalCount = 100  # 驼峰命名
valid = True  # 缺少 is 前缀
max_retry = 3  # 常量应大写
```

### 函数命名

```python
# ✅ 正确
def get_user_info(user_id):
    pass

def calculate_total_price(items):
    pass

def is_user_logged_in():
    pass

# ❌ 错误
def getUserInfo(user_id):  # 驼峰命名
    pass

def func1(items):  # 无意义名称
    pass
```

### 类命名

```python
# ✅ 正确
class UserManager:
    pass

class DatabaseConnection:
    pass

# ❌ 错误
class user_manager:  # 应使用大驼峰
    pass
```

## 代码格式

### 缩进

```python
# ✅ 正确：4 个空格
def hello():
    print("Hello")
    if True:
        print("World")

# ❌ 错误：Tab 或 2 个空格
def hello():
  print("Hello")  # 2 个空格
```

### 空行

```python
# ✅ 正确
class MyClass:
    """类文档字符串"""
    
    def __init__(self):
        pass
    
    def method1(self):
        pass
    
    def method2(self):
        pass

# ❌ 错误：缺少空行
class MyClass:
    def __init__(self):
        pass
    def method1(self):  # 缺少空行
        pass
```

### 行长度

```python
# ✅ 正确：不超过 100 字符
long_string = "这是一个很长的字符串，但是如果超过 100 字符就应该换行，保持代码可读性"

# ❌ 错误：超过 100 字符
long_string = "这是一个很长的字符串，但是如果超过 100 字符就应该换行，保持代码可读性，不要写在一行"
```

## 注释规范

### 文档字符串

```python
# ✅ 正确
def calculate_price(items, discount=0):
    """
    计算商品总价
    
    Args:
        items (list): 商品列表
        discount (float): 折扣，0-1 之间，默认为 0
    
    Returns:
        float: 总价
    
    Raises:
        ValueError: 当折扣不在 0-1 之间时
    """
    if not 0 <= discount <= 1:
        raise ValueError("折扣必须在 0-1 之间")
    
    total = sum(item['price'] for item in items)
    return total * (1 - discount)

# ❌ 错误：缺少文档字符串
def calculate_price(items, discount=0):
    total = sum(item['price'] for item in items)
    return total * (1 - discount)
```

### 行内注释

```python
# ✅ 正确
user_id = get_user_id()  # 获取用户 ID
if user_id is None:  # 用户未登录
    return redirect('/login')

# ❌ 错误：无意义注释
user_id = get_user_id()  # 获取 user_id
if user_id is None:  # 如果为 None
    return redirect('/login')  # 重定向
```

## 异常处理

### 捕获异常

```python
# ✅ 正确
try:
    result = risky_operation()
except ValueError as e:
    logger.error(f"值错误：{e}")
    return None
except Exception as e:
    logger.exception(f"未知错误：{e}")
    raise

# ❌ 错误
try:
    result = risky_operation()
except:  # 捕获所有异常
    pass  # 忽略异常
```

### 自定义异常

```python
# ✅ 正确
class BusinessError(Exception):
    """业务异常基类"""
    pass

class UserNotFoundError(BusinessError):
    """用户未找到异常"""
    pass

# ❌ 错误
def process_user(user_id):
    if user_id is None:
        raise Exception("用户错误")  # 使用通用异常
```

## 安全规范

### 敏感信息

```python
# ✅ 正确
import os
API_KEY = os.environ.get('API_KEY')  # 从环境变量读取

# ❌ 错误
API_KEY = "sk-1234567890abcdef"  # 硬编码密钥
```

### SQL 注入

```python
# ✅ 正确：参数化查询
cursor.execute("SELECT * FROM users WHERE id = ?", (user_id,))

# ❌ 错误：字符串拼接
cursor.execute(f"SELECT * FROM users WHERE id = {user_id}")  # SQL 注入风险
```

### 密码存储

```python
# ✅ 正确：哈希存储
import bcrypt
hashed = bcrypt.hashpw(password.encode(), bcrypt.gensalt())

# ❌ 错误：明文存储
password = "123456"  # 明文密码
```

## 性能规范

### 循环优化

```python
# ✅ 正确：列表推导式
squares = [x**2 for x in range(100)]

# ❌ 错误：低效循环
squares = []
for x in range(100):
    squares.append(x**2)
```

### 数据库查询

```python
# ✅ 正确：批量查询
users = User.query.filter(User.id.in_(user_ids)).all()

# ❌ 错误：N+1 查询
for user_id in user_ids:
    user = User.query.filter_by(id=user_id).first()
```

## 测试规范

### 单元测试

```python
# ✅ 正确
import pytest

def test_calculate_price():
    items = [{'price': 100}, {'price': 200}]
    assert calculate_price(items) == 300

def test_calculate_price_with_discount():
    items = [{'price': 100}]
    assert calculate_price(items, discount=0.1) == 90

# ❌ 错误：没有测试
# 完全没有测试代码
```

### 测试覆盖

```python
# ✅ 正确：覆盖边界条件
def test_empty_items():
    assert calculate_price([]) == 0

def test_invalid_discount():
    with pytest.raises(ValueError):
        calculate_price([], discount=1.5)

# ❌ 错误：只测试正常情况
def test_normal_case():
    assert calculate_price([{'price': 100}]) == 100
```
