## 专题 1：安全开发

### 1.1 密钥管理

**❌ 不要**
```python
API_KEY = "sk-1234567890abcdef"
PASSWORD = "admin123"
```

**✅ 要**
```python
import os
API_KEY = os.environ.get("API_KEY")
PASSWORD = os.environ.get("PASSWORD")
```

### 1.2 SQL 查询

**❌ 不要**
```python
cursor.execute(f"SELECT * FROM users WHERE id = {user_id}")
```

**✅ 要**
```python
cursor.execute("SELECT * FROM users WHERE id = ?", (user_id,))
```

### 1.3 用户输入

**❌ 不要**
```python
html = f"<div>{user_input}</div>"  # XSS 风险
```

**✅ 要**
```python
import html
html = f"<div>{html.escape(user_input)}</div>"
```

## 专题 2：API 设计

### 2.1 统一响应

**❌ 不要**
```python
return {"ok": True, "user": user}
return {"status": "success", "data": user}
return {"code": 0, "result": user}
```

**✅ 要**
```python
def success(data=None, message="成功"):
    return jsonify({"code": 200, "message": message, "data": data})

def error(code=400, message="失败"):
    return jsonify({"code": code, "message": message, "data": None})
```

### 2.2 分页 API

**❌ 不要**
```python
# 不分页返回全量数据，数据量大时接口崩溃
return success(users)
```

**✅ 要**
```python
@app.route("/api/users")
def list_users():
    page = int(request.args.get("page", 1))
    size = min(int(request.args.get("size", 20)), 100)
    offset = (page - 1) * size
    total = db.query("SELECT COUNT(*) FROM users")[0][0]
    rows = db.query("SELECT * FROM users LIMIT ? OFFSET ?", (size, offset))
    return success({
        "items": rows,
        "total": total,
        "page": page,
        "size": size,
        "pages": (total + size - 1) // size
    })
```

## 专题 3：数据库

### 3.1 批量操作

**❌ 不要**
```python
for user in users:
    db.execute("INSERT INTO users VALUES (?, ?)", (user.name, user.age))
```

**✅ 要**
```python
data = [(u.name, u.age) for u in users]
db.executemany("INSERT INTO users VALUES (?, ?)", data)
conn.commit()
```

### 3.2 N+1 查询

**❌ 不要**
```python
orders = get_orders()
for order in orders:          # 1 次查询
    user = get_user(order.user_id)  # N 次查询
```

**✅ 要**
```python
orders = get_orders()
user_ids = [o.user_id for o in orders]
users = get_users_by_ids(user_ids)  # 2 次查询
users_dict = {u.id: u for u in users}
```

## 专题 4：错误处理

### 4.1 异常捕获

**❌ 不要**
```python
try:
    result = api_call()
except:
    pass  # 静默忽略
```

**✅ 要**
```python
try:
    result = api_call()
except ValueError as e:
    logger.warning(f"参数错误：{e}")
    return error(400, str(e))
except Exception as e:
    logger.exception(f"未知错误：{e}")
    return error(500, "服务器错误")
```

### 4.2 资源释放

**❌ 不要**
```python
f = open("data.txt")
content = f.read()
# 忘记关闭文件
```

**✅ 要**
```python
with open("data.txt") as f:
    content = f.read()
```

## 专题 6：日志

### 6.1 日志级别

**❌ 不要**
```python
print("用户登录成功")
print(f"错误：{e}")
```

**✅ 要**
```python
import logging
logger = logging.getLogger(__name__)

logger.info(f"用户 {user_id} 登录成功")
logger.warning(f"登录失败：{e}")
logger.error(f"系统错误：{e}", exc_info=True)
```

### 6.2 结构化日志

**❌ 不要**
```python
logger.info(f"用户 {user_id} 在 {time} 登录，IP={ip}")
```

**✅ 要**
```python
logger.info(json.dumps({
    "event": "user_login",
    "user_id": user_id,
    "ip": ip,
    "timestamp": datetime.now().isoformat()
}))
```
