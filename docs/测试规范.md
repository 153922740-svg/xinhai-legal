## 单元测试规范

### 测试框架

```python

# 安装：pip install pytest pytest-cov

# 测试文件命名
test_user_api.py          # ✅ 正确
user_api_test.py          # ❌ 错误
test_user.py              # ✅ 正确
```

### 测试用例结构

```python
import pytest
from api.user_api import UserAPI

class TestUserAPI:
    """用户 API 测试类"""
    
    def setup_method(self):
        """每个测试前的准备工作"""
        self.api = UserAPI()
        self.test_user = {'name': '测试用户', 'phone': '13800138000'}
    
    def teardown_method(self):
        """每个测试后的清理工作"""
        self.api.cleanup()
    
    def test_create_user_success(self):
        """测试创建用户成功"""
        # Arrange (准备)
        user_data = self.test_user
        
        # Act (执行)
        result = self.api.create_user(user_data)
        
        # Assert (断言)
        assert result['code'] == 200
        assert result['data']['name'] == '测试用户'
        assert 'id' in result['data']
    
    def test_create_user_empty_name(self):
        """测试创建用户 - 空名字"""
        # Arrange
        user_data = {'name': '', 'phone': '13800138000'}
        
        # Act
        result = self.api.create_user(user_data)
        
        # Assert
        assert result['code'] == 400
        assert '名字不能为空' in result['message']
    
    def test_create_user_invalid_phone(self):
        """测试创建用户 - 无效手机号"""
        # Arrange
        user_data = {'name': '测试用户', 'phone': '12345'}
        
        # Act
        result = self.api.create_user(user_data)
        
        # Assert
        assert result['code'] == 400
        assert '手机号格式错误' in result['message']
```

## 持续集成

### CI 测试流程

```yaml
# .github/workflows/test.yml
name: Test

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v2
    
    - name: Set up Python
      uses: actions/setup-python@v2
      with:
        python-version: 3.11
    
    - name: Install dependencies
      run: pip install -r requirements.txt
    
    - name: Run tests
      run: pytest --cov=. --cov-fail-under=70
    
    - name: Upload coverage
      uses: codecov/codecov-action@v2
```
