**状态**: 服务器上未找到证书文件

**解决方案**:

1. 登录 https://pay.weixin.qq.com
2. 账户中心 → API 安全 → 申请 API 证书
3. 下载证书文件（apiclient_cert.pem 和 apiclient_key.pem）
4. 上传到服务器：`/home/admin/xinhai-legal/cert/`
