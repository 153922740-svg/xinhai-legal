## 📌 使用说明

### 发现问题时
1. 在「待修复问题」表中添加新行
2. 记录问题描述、文件路径、优先级
3. 存入 Hindsight 记忆系统（关键词：代码问题 + 文件名）

### 修复问题时
1. 从「待修复问题」移动到「已修复问题」
2. 填写修复方案和 Git 提交 ID
3. 更新「代码修改记录」
4. 更新 Hindsight 记忆状态

### 会话开始时
1. 自动加载本文件
2. recall Hindsight 记忆中的问题记录
3. 向总裁汇报当前问题清单
